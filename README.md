# 💰 Money – Expense Tracker App

A modern, cross-platform expense tracking application built with Flutter.
Track daily expenses, monitor spending behavior, and manage your budget with a fast, clean, offline-first experience.

---

# 🚀 Features

## 📊 Dashboard

* **Total Expense Overview** — See cumulative spending instantly
* **Monthly & Daily Stats** — Quick insight into short-term patterns
* **Recent Transactions** — Last 5 expenses with fast navigation
* **Real-time Updates** — Fully reactive UI using signals

---

## 💰 Expense Management

* **Add Expenses** — Simple note + amount input
* **Edit & Delete** — Complete CRUD functionality
* **Search** — Instant filtering by notes
* **Sorting Options**

  * By date
  * By amount
  * By description

---

## 🎨 User Experience

* **Material Design 3**
* **Dark / Light Mode Toggle**
* **Responsive Layout** (mobile, tablet, desktop, web)
* **Smooth Bottom Navigation**
* **Offline-First Architecture**

---

## 💾 Data & Architecture

* **ObjectBox Local Database**
* **Signals-based Reactive State**
* **Instant UI Consistency**
* **No Internet Required**
